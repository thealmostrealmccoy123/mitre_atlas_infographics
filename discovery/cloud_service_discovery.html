Cloud Service Discovery

Cloud Service Discovery is a tactic where adversaries attempting to enumerate cloud services ranging from infrastructure and platforms to AI-as-a-service (AIaaS) running on a system after gaining initial access. Attackers utilize tools and APIs, such as the Microsoft Graph API, to identify available resources, relationships, security services like AWS GuardDuty, and logging configurations to orient themselves within the environment. In the context of AI, this involves using specific tools to check credentials and list accessible AI models across providers like AWS Bedrock and OpenAI to determine resource quotas and logging settings.
