AI Agent Tool Invocation

Adversaries exploit access to AI agents to invoke connected tools and services, such as APIs or internal data sources, that possess permissions unavailable to the direct user. By manipulating these agents to execute unauthorized commands or API calls, attackers can escalate their privileges and perform actions like retrieving sensitive information or bypassing security controls. Case studies illustrate this tactic involves abusing the agent's trusted internal context or specific plugins, such as enterprise search functionalities, to carry out malicious activities with the agent's elevated access rights.
