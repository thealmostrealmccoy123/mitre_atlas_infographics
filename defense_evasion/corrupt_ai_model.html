Corrupt AI Model

The Corrupt AI Model technique involves an adversary purposefully damaging a malicious AI model file to prevent it from being successfully deserialized, which allows the file to evade detection by model scanners like Picklescan. Although the corruption ultimately causes the deserialization process to fail, the attack is designed so that the embedded malicious code executes before the failure occurs. This tactic falls under the broader category of Defense Evasion, as it specifically aims to bypass AI-enabled security software while delivering payloads such as reverse shells to compromise a victim's system.
